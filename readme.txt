This plugin provides a button for WP-Google Syntax Highlighter in TinyMCE editor.

== Description ==

If you use the plugin "WP-Google Syntax Highlighter", you should install this plugin.
WP-Google Syntax Highlighter is a plugin to highlight code.
Our plugin provides a button for WP-Google Syntax Highlighter in TinyMCE editor.
So, it becomes easy to use the Google Syntax Highlighter.

== Installation ==

1. Upload the directory "wp-gsh-button" to the "/wp-content/plugins/" directory
1. Activate the plugin through the "Plugins" menu in WordPress
1. Also activate the WP-Google Syntax Highlighter

== Frequently Asked Questions ==

= Are there the any one necessary for another? =

Yes.
You also have to install WP-Google Syntax Highlighter.
